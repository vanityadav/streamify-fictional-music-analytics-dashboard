# Streamify Service

## Music Analytics Dashboard

## Getting Started

### Tech

- Next.js 15
- React 19
- Typescript
- React Context - State Management

### Installation

- Install the dependencies and start the server.

```
npm i

npm run dev

npm run start
```

### Tests

- React testing library is not supported on react 19
